# Installation

## Stable release

To install ml_template, run this command in your terminal:

```sh
uv add ml_template
```

Or if you prefer to use `pip`:

```sh
pip install ml_template
```

## From source

The source files for ml_template can be downloaded from the [Github repo](https://github.com/basnied/ml_template).

You can either clone the public repository:

```sh
git clone git://github.com/basnied/ml_template
```

Or download the [tarball](https://github.com/basnied/ml_template/tarball/master):

```sh
curl -OJL https://github.com/basnied/ml_template/tarball/master
```

Once you have a copy of the source, you can install it with:

```sh
cd ml_template
uv pip install .
```
